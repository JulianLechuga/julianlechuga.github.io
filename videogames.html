Videogames Project App

This project consists of a SPA (Single Page Application) where, based on the Videogames API provided by RAWG, I can display a catalog of videogames and allow users to create their own, storing them in a database.

It's main functionalities are:

  • Display a list of video games.
  • Search videogames by name
  • Create video games.
  • Filter: alphabetically, by rating, by genre and if they were created by users or brought from the API.
  • Paginated.

The technologies I used to develop this app are:

  • JavaScript
  • Database: PostgreSQL.
  • Back-End: NodeJs, ExpressJs, Sequelize.
  • Front-End: React, Redux, CSS.
  • Deploy: Heroku and Vercel.
  • Version Control: Git/GitHub.
